## Octal

O base-8, es la representación de utilizar los dígitos del 0 al 7. Antiguamente era un sistema muy utilizado en informática.

### Python

En Python también se puede trabajar en octal, para ello, pon como prefijo `0o`, y si quieres ver la representación en octal de un número, usa la función `oct()`:

```python
>>> 0o7
7
>>> 0o17
15
>>> 0o172
122
>>> oct(122)
'0o172'
>>> oct(0b1000110)
'0o106'
```
